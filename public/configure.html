Install in Nuvio

  1. Generate and copy the manifest URL above.
  2. In Nuvio on iOS or TV, open Settings → Addons and add that URL.
  3. Disable the original wrapped catalog addons to avoid duplicate catalog rows. If a stream addon was not included above, leave it installed normally.

Only media with an IMDb ID can use this BetterPosters URL format. Configuration tokens contain the addon URLs in reversible form; do not include URLs you consider secret on a shared deployment.
